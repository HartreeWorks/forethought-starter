# Forethought typography

Typography specifications from the Forethought Brand Guidelines (February 2025).

## Primary typeface: Signifier

Signifier is a refined, Brutalist response to 17th century typefaces, giving the brand a sense of academic credibility.

**Used for:** Headlines, introductory text, body copy, captions

### Weights

| Weight | Use |
|--------|-----|
| **Extralight** | Large headlines (50pt), headlines (35pt), captions (15pt ALL CAPS) |
| **Light** | Body text (18pt) |

### Sizing

| Element | Weight | Size | Line height |
|---------|--------|------|-------------|
| Large headlines | Extralight | 50pt | 115% |
| Headlines | Extralight | 35pt | 120% |
| Body | Light | 18pt | 150% |
| Captions | Extralight | 15pt (ALL CAPS) | 135% |

## Secondary typeface: TT Hoves

TT Hoves is a neutral sans-serif with a visual lack of contrast, creating a timeless feel that supports Signifier for key details.

**Used for:** Smaller headlines, navigation, tags, annotations

### Weights

| Weight | Use |
|--------|-----|
| **Light** | Supplementary text |
| **Regular** | Navigation (16pt), tags (15pt) |
| **Medium** | Small headlines (22pt) |

### Sizing

| Element | Weight | Size | Line height |
|---------|--------|------|-------------|
| Large headlines | Regular | 50pt | 115% |
| Small headlines | Medium | 22pt | 120% |
| Navigation | Regular | 16pt | 110% |
| Tags | Regular | 15pt | 103% |

## Fallback fonts

When Signifier and TT Hoves are unavailable (e.g. in Canva, Google Docs), use these approved alternatives:

### System fonts

| Brand font | Fallback | Weights |
|------------|----------|---------|
| Signifier | Garamond | Regular, Semibold |
| TT Hoves | Helvetica Neue | Regular, Medium |

### Google fonts (for Canva and Google Docs)

| Brand font | Fallback | Weights |
|------------|----------|---------|
| Signifier | **Cormorant Garamond** | Light, Regular |
| TT Hoves | **Inter** | Regular, Medium |

**In Canva:** Always use Cormorant Garamond and Inter as these are readily available and closest to the brand fonts.

## Typography in diagrams

### Text hierarchy

| Level | Font | Weight | Colour |
|-------|------|--------|--------|
| Main title | Cormorant Garamond | Regular | Charcoal #2F2A26 |
| Section headers | Cormorant Garamond | Light | Charcoal #2F2A26 |
| Body/descriptions | Cormorant Garamond | Light | Charcoal #2F2A26 |
| Labels/annotations | Inter | Regular | Charcoal #2F2A26 or Dark Grey #757371 |
| Data values | Inter | Regular | Charcoal #2F2A26 |

### Size recommendations for diagrams

| Element | Size range |
|---------|------------|
| Main title | 24-36pt |
| Section headers | 18-24pt |
| Body text | 14-18pt |
| Labels/annotations | 10-14pt |
| Axis labels | 10-12pt |

### Legibility guidelines

- **Minimum size:** 10pt for any text that must be readable
- **Contrast:** Always use Charcoal #2F2A26 on Off-white #FBFAF4 background
- **Line length:** Aim for 50-75 characters per line for body text
- **Avoid:** All caps except for captions and tags

## Colour with typography

| Element | Text colour |
|---------|-------------|
| Primary text | Charcoal #2F2A26 |
| Secondary/muted text | Dark Grey #757371 |
| Accent text (sparingly) | Orange #FA7248 |
| Text on Charcoal bg | Off-white #FBFAF4 |
| Text on Orange bg | Off-white #FBFAF4 or Black |

**Note:** Never use Orange for paragraphs of text. Use it only for highlights, tags, and interactive elements.

## Web/digital specifications

For web applications:
- Signifier → Cormorant Garamond (Google Fonts)
- TT Hoves → Inter (Google Fonts)

For system defaults (when custom fonts can't load):
- Signifier → Garamond
- TT Hoves → Helvetica Neue
