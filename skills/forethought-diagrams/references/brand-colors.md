## Usage guidelines

### Background colour (CRITICAL)

**Always use #FBFAF4 for diagram backgrounds.**

Rationale:
- Transparent backgrounds display as dark in dark mode, making text invisible
- Pure white (#FFFFFF) creates a visible mismatch with the website background
- The off-white (#FBFAF4) matches the Forethought website and creates a cohesive look

### Text colours

- **Primary text:** Charcoal #2F2A26
- **Secondary text:** Dark Grey #757371
- **Never use:** Orange for body text (only for accents)

### Accent colours

- Use Orange #FA7248 sparingly for highlights, buttons, and key elements
- Illustrations should only use Orange on Off-white background
- Avoid using Orange for large panels or backgrounds

### Data visualisation order

When creating charts with multiple data series, use colours in this order:
1. Blue #4988A9
2. Green #43B85D
3. Purple #8B61A6
4. Yellow #E6C90A
5. Orange #FF774C
6. Red #A7313E

For accessibility-critical contexts, use the Accessibility variants instead.

## Print specifications

For print applications:
- Use Pantone 164 C for Orange
- Use Pantone 2387 C for Charcoal
- For other colours, use CMYK values from the brand guidelines PDF
