# Platforms and tools

## Contentful (website CMS)

The Forethought website is managed through Contentful.

**URL**: https://app.contentful.com/

**Publishing an article:**
1. Go to Content → filter for Articles
2. Find your article
3. Set "Unlisted" to "No"
4. Check publication date
5. Press green "Publish" button

**Featuring an article:**
1. Content → Series → Featured articles
2. Add your piece at the top

**Adding Forum/LW links:**
- In the Article, scroll to EA Forum and LessWrong fields near bottom
- Add the post URLs

## Substack (ForeWord newsletter)

**URL**: https://forethoughtnewsletter.substack.com/

**Access**: All researchers are Contributors on the Forethought Substack via their own Substack accounts.

**Creating a post:**
1. Go to https://forethoughtnewsletter.substack.com/publish/
2. Create new text post
3. Set Section to "ForeWord"

**Importing content:**
- Copy-paste from Google Docs (may need cleanup)
- Or use online Markdown to Rich Text converter

**Adding coauthors:**
- Forethought team members appear in dropdown
- External people: search for Substack account or enter email
- External coauthors need to accept invite before showing

**Standard footer:**
```
This article was created by [Forethought](https://www.forethought.org/about). See all of our research on [our website](https://www.forethought.org/research).
```

**Preview image:**
- Settings → Social preview
- Add the chosen image

**SEO (optional):**
- Settings → SEO Options
- Title: under 60 characters
- Description: 50-160 characters

**Finding drafts:**
https://forethoughtnewsletter.substack.com/publish/posts/drafts

## Typefully (social media)

**URL**: https://typefully.com/

**Purpose**: Post to multiple Forethought social accounts simultaneously.

**Accounts**: Twitter, LinkedIn, Bluesky, Threads

**Creating a thread:**
1. Log in to Typefully
2. Draft thread
3. Select platforms (default: all)
4. Quote-tweets: paste URL, will offer to convert
5. Check versions for each platform before posting

**Notes:**
- Quote-tweets appear as links on non-Twitter platforms
- LinkedIn shows whole thread as one post with images at end
- Can't retweet natively — only quote-tweet

**Alternative for Twitter retweets:**
Log in to Forethought Twitter directly (details in Social media & Forum accounts doc)

## EA Forum

**URL**: https://forum.effectivealtruism.org/

**Forethought account**: Login details in 1Password team vault

**Creating a post:**
1. Log in to Forethought account
2. Create as "Linkpost" to website or Substack
3. Add yourself + coauthors as coauthors
4. Import content:
   - "Import Google doc" function in editor
   - Or download GDoc as Markdown and paste

**Standard footer:**
```
This post was created by [Forethought](https://www.forethought.org/about). Read the original on our [website/Substack] [link].
```

**Notes:**
- Forethought account used for posting, but individual coauthors still show
- No LessWrong crosspost UI when posted from Forethought account

## LessWrong

**URL**: https://www.lesswrong.com/

**Account**: Use your own personal account (org accounts not allowed)

**Creating a post:**
1. Log in to your account
2. Create as "Linkpost"
3. Import content same as EA Forum

**Finding drafts:**
Hover username → "My drafts"

**Notes:**
- Posts always from personal account
- Can crosspost to EA Forum, but loses coauthor display
